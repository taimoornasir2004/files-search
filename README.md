# Files Search

A desktop file search application built in C#, allowing users to search for documents either by title or by content, with configurable match modes.

## 🎮 Features

- **Search by Title** — find files based on their filename/title
- **Search in Document** — find files based on content within the document
- **Match Modes:**
  - **Exactly** — matches the exact search term
  - **Similar** — matches similar/related terms
- **Search Term Input** — dialog box to enter the search query
- **Clickable Results** — matched files are displayed as clickable links that open the file directly

## 🛠️ Tech Stack

- **Language:** C#
- **Framework:** .NET (Windows Forms)
- **IDE:** Visual Studio

## ⚙️ How to Run

1. Clone the repository
2. Open the `.sln` file in Visual Studio
3. Build and run the project (F5)
4. Choose search mode (by Title or in Document), select match type (Exactly/Similar), enter your search term, and click Search
